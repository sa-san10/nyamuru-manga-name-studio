# 行動規範

このプロジェクトは、誰もが安心して参加できる場を目指します。

## 期待すること

- 相手への敬意を持ったコミュニケーション
- 建設的なフィードバック
- 経験・立場・背景の違いの尊重

## 許容しないこと

- ハラスメント、個人攻撃、差別的な言動
- 他者の個人情報の無断公開
- 参加を妨げる継続的な妨害行為

## 適用範囲

Issue、Pull Request、その他このプロジェクトに関わるすべての場に適用されます。

## 報告

問題のある行為を見た・受けた場合は、公開の場に書きにくければリポジトリの「Security」タブのプライベート報告機能、またはGitHub経由でメンテナー（[sa-san10](https://github.com/sa-san10)）へ連絡してください。メンテナーは内容を確認し、注意・投稿の削除・参加制限など適切な対応を行います。
